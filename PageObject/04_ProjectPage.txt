# Positive Tests

Title: New blank project
Description: Verifies that user can create a new blank project with valid data
Precondition: Browser is open, user is logged in GitLab
Steps:
	Navigate to Projects page		Projects_Url
	Select New project		
	Select Create Blank project	
	Input Project name				BlankProjectName
	Select Project Group or User	ProjectGroup
	Verify or Edit Project slug		BlankSlug
	Select Visibility Level			VisibilityPrivate
	Select Project Configuration	ConfigurationREADMEPositive
	Submit Create project Form
	Verify Blank Project is created
		
Postcondition: Blank Project is successfully created and added to the user's project list


Title: New template project
Description: Verifies that user can create a new template project with valid data
Precondition: Browser is open, user is logged in GitLab
Steps:
	Navigate to Projects page		Projects_Url
	Select New project		
	Select Create from template	
	Select template					TemplateSelection
	Input Project name				TemplateProjectName
	Select Project Group or User	ProjectGroup

Postcondition: Template project is successfully imported and added to the user's project list


Title: Update project name
Description: Verifies that the user can update the name of a project
Precondition: Browser is open, user is logged in GitLab
Steps: 
	Navigate to Template Project page	TemplateProject_Url

	Select Edit

	Submit by Save changes
	Verify project name is updated
	Verify success message "Project was successfully updated" is displayed
	
Postcondition: The template project's name is successfully updated


Title: Delete project
Description: Verifies that the user can delete a project
Precondition: Browser is open, user is logged in GitLab
Steps:
	Navigate to Template Project page	TemplateProject_Url
	Select options menu
	Select Delete	
	Input given text					TemplateUrlPart
	Submit by Yes, delete project		
	Verify template project has been moved to pending deletion
	Verify message "Templatetest moved to pending deletion" is displayed
	Navigate to inactive projects		Inactive_Url
	Select project awaiting deletion	TemplateProjectName
	Select options menu	
	Select delete permanently		
	Input given text					DeletePermanentlyUrlPart
	Select permanent deletion option	
	Submit by Yes, delete project
	Verify template project is being deleted
	Verify message "Templatetest-deletion_scheduled-81646652 is being deleted" is displayed

Postcondition: The Template project has been successfully deleted

Title: Transfer project

Precondition: Browser is open, user is logged in GitLab

Postcondition: The project has been successfully transferred
