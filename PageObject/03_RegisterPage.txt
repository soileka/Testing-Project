# Positive Tests

Title: Valid data		
Description: Verifies that user can register with valid data
Precondition: Browser is open
Steps:
	Navigate to Sign Up page		SignUp_Url
	Input First name 				FirstName
	Input Last name					LastName
	Input User name					UserName
	Input Email						Email
	Input Password					Password
	Submit Registration Form
	Input Email Verification Code	VerificationCode
	Submit Email Verification
	Verify that "Verification was successful" message is shown
	Verify that registration process moves on to page that includes selections for how the user plans to use GitLab
	Select Role						Role
	Select Reason for SignUp		SignUpReason
	Select What to do in GitLab		WhatToDo
	Select Who uses GitLab			GitLabUser
	Submit selections with Continue button
	Verify that registration process moves on to page that asks the user to create or import a project
	Input Group name				GroupName
	Input Project name				ProjectName
	Submit Create Project Form
	Verify that the user account has been created and GitLab is accessible

Postcondition: User account is successfully created and user is logged in
